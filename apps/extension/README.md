# LinguaAI Browser Extension

Chrome extension để học từ vựng ngay khi lướt web.

## Cài đặt (Developer Mode)

1. Mở Chrome → `chrome://extensions/`
2. Bật **Developer mode** (góc trên phải)
3. Nhấn **Load unpacked**
4. Chọn thư mục `apps/extension`

## Tính năng

- **Highlight từ** trên bất kỳ trang web nào → tooltip xuất hiện ngay
- **Tra từ** với IPA, từ loại, nghĩa, ví dụ
- **Lưu flashcard** về local storage
- **Context menu** (chuột phải) → "LinguaAI: Tra từ"
- **Popup** với search box, danh sách flashcard đã lưu
- **Sync settings** ngôn ngữ học, ngôn ngữ mẹ đẻ

## Cấu hình

Mở popup → tab ⚙️ Cài đặt → nhập URL của LinguaAI app.

## Icons

Cần tạo thư mục `icons/` với các file:
- `icon16.png` (16×16)
- `icon48.png` (48×48)  
- `icon128.png` (128×128)

Có thể dùng emoji 🧠 hoặc logo LinguaAI.
